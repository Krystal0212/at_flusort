## 0.0.1

- Initial release of `at_flusort`.
- Implemented sorting algorithms:
  - Bubble Sort
  - Selection Sort
  - Insertion Sort
  - Merge Sort
  - Quick Sort
  - Heap Sort
  - Radix Sort
  - Bucket Sort
- Supports generic types: `int`, `double`, `String`, `DateTime`, `Color`, custom classes (e.g. Firebase Document, Coordinate, etc.)
